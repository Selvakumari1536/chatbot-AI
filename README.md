A fully functional, browser-based medical chatbot that provides general health advice, diet recommendations, and daily routines for common conditions.

## Features

- **Interactive Chat Interface**: Real-time chat with simulated AI responses
- **Common Conditions**: Quick access buttons for diabetes, hypertension, arthritis, weight loss, etc.
- **Health Tips**: Daily health recommendations with visual icons
- **Symptom Checker**: Basic guidance for common symptoms
- **Emergency Information**: Quick access to emergency contacts
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **No Backend Required**: Runs entirely in the browser
- **Chat History**: Saves conversations locally

## How to Use

1. **Open the Application**:
   - Simply open `index.html` in any modern web browser
   - No installation or server required

2. **Using the Chatbot**:
   - Click on any condition in the left sidebar
   - Type your health questions in the chat box
   - Use quick suggestion buttons
   - Check symptoms using the symptom checker

3. **Features**:
   - Left Sidebar: Condition selection and settings
   - Center: Main chat interface
   - Right Sidebar: Health tips and emergency contacts
